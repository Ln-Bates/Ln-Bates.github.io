JS 数组方法-2

2017.09.13 bates 技术  热度 ℃

位置方法

arr.indexOf(); // 正序检错, 返回对应索引, 没有返回-1
arr.lastIndexOf(); // 倒叙检索, 返回对应索引, 没有返回

迭代方法

arr.every(); // 遍历每项, 每一项返回true即返回true
arr.some(); // 遍历每项, 某一项返回true即返回true
arr.map(); // 返回新数组
arr.filter(); // 遍历每项, 返回符合条件的组成的新数组
arr.forEach(); // 迭代数组没有返回值
